# Boilerplate Vue App SPA

## Module Include

* Vue-Cli
* Vue-Router
* Vuex (customize)
* Vuetify (CSS Library)
* Eslint & Prettier (Linter)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
